How this works

  1. Click Open Account DB and pick SWGTCG-Standalone\_ext\server\swgtcg.db. This is the account database the game logs into — editing it here changes what you own and which decks you can play in Skirmish. Works in Microsoft Edge / Chrome.
  2. The launcher logs in as StandAloneUser — the single built-in account (it owns all four starter decks: Imperial / Jedi / Rebel / Sith). Pick it under Account.
  3. Collection tab: set how many copies of each card you own. Deck Builder: pick an avatar, add main-deck cards + quests, Save deck.
  4. Scenarios tab: view saved single-player progress, Reset it, or Grant a scenario complete (a scenario becomes grantable after you’ve played it once).
  5. Boosters tab: pick a set and Open Booster Pack to add random cards to your collection, shown with art, text and a rarity glow. Card art isn’t shipped — run the launcher’s [7] Extract card art once to build it from your own cards.rcc.
  6. Click Save to game to write the database back. Important: stop the game and the login server first (launcher option 5), or the save can be lost / conflict. Then run Play again — your changes are read at login.
  7. No folder access in your browser? Use Import DB / the Save button downloads the edited swgtcg.db — copy it over _ext\server\swgtcg.db.

Deck rules of thumb (Standard): 1 avatar, ~50 main cards, up to 4 quests, at most 4 copies of a card. The builder shows warnings but won't stop you experimenting.
